Web development cheat sheet

Git version control

Configuration initiale

git config --global user.name "your name"
git config --global user.email "your@email.com"

Initialisation repo dossier de travail

cd ~/my_working_directory
git init

Création du repo sous Github

Cliquer sur New Repository
Copier le lien de l'option "push an existing repo from the command line"

git remote add origin https://github.com/xxxxx/my_repo.git
git push -u origin master

First commit

git add .
git commit -m "commit description"
git push origin master

Contrôle de l'état du repo

git status
git log

Gestion des branches

Afficher les branches existantes

git branch

Créer une branche

git branch my_new_branch

Changer de branche

git checkout my_new_branch

Merge la branche dev dans master

git checkout master
git merge dev
